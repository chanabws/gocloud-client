Component 1: Leadership Plan/Scope


IBM will work with ICT SMEs to deliver a Project Plan that is signed off by ICT SME Management Level Project Sponsor.

As part of the Go Cloud Native: IBM programme, IBM will organize a one full day IBM Go Cloud Native Seminar Workshop specially for Leadership Team from all ICT SMEs in the same batch.

As part of the Go Cloud Native: IBM programme, IBM will have one full day deep-dive workshop with each ICT SME enrolled in the programme.

Component 2: Acquire Techincal Skills


IBM Cloud Native Bootcamp
IBM propose IBM Cloud Native Developer Course to provide the foundation knowledge and skills for ICT SME’s technical staff to be skill up to support the ICT SME adoption of Cloud Native, Microservices and DevOps.

IBM Cloud Native ICP Bootcamp
This course delivers expert–led training on how to approach cloud application development on IBM Cloud ICP.

IBM Cloud Garage Architect Bootcamp
IBM propose IBM Cloud Native Architect course to provide the intermediate knowledge and skills for ICT SME’s technical architect and lead developer to be skill up to support the architecture, design, and process of developing, deploying and maintaining cloud-native applications on Cloud. This skill will help ICT SME strategic technical planning and enhancement of their offering with Cloud Native, Microservices and DevOps.

Component 3: Transform Monolithic Solution


IBM will provide a coaching approach to guide ICT SMEs to complete the scoped project to develop\transform an application to Cloud Native architecture using Micro-services and DevOps practices during the 3 months duration.
